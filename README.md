# Video to Image
## Video format 
- mov,mp4,m4a,3gp,3g2,mj2
## Image format 
- PNG, JPG

## Install
    pip install -r requirement.txt
If you had permission error.

    python -m pip install "<module>" --user

## CLI
    python convert.py [video path] [result path] [step] [image type : PNG / JPG]

## GUI
    python main.py
